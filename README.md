# ROW/COL Math

## Context

* **Origin** Developed as part of a course in C at Karlstad University.
* **Objective** A console app that allows the user to do very basic matrix calculations.
* **Status** 🟢 Complete/Functional

---

## Systems Architechture

* **Logic** Single file procedural.
* **Tech Stack** C

---

## Functionality

* Generate a randomized matrix
* Sort the matrix (lowest -> highest)
* Calculate avarage/median/min/max values
* Search a number in the matrix and get its' position
* Calculate the sum of the rows/cols and total

---

## Setup & Usage

1. Clone the repository
2. Open in IDE of your choice
3. Do some basic matrix calculation

---

## Learning Outcomes

* Defining functions in C
* Building basic console menus in C
* Doing basic matrix calculation
